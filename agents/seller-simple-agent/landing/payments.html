How Agent Payments Work

An AI agent discovers, pays, and consumes ComplianceShield in four autonomous steps.

1
🔍

Discovery

Agent fetches /.well-known/agent.json to discover ComplianceShield's capabilities, pricing, and payment requirements.

➡
2
🎫

Token Acquisition

Agent acquires an x402 payment token from Nevermined by purchasing credits against the plan ID.

➡
3
📨

Paid Request

Agent sends compliance scan request with payment-signature header containing the x402 access token.

➡
4
✅

Validate & Settle

ComplianceShield validates the token via Nevermined, runs the scan, deducts credits, and returns the report.

Tool Pricing

Each tool costs a fixed number of credits per invocation. Credits are deducted atomically on each call.

Tool Credits Protocol Description
quick_scan ⚡ 1 A2AMCP Fast compliance check against FTC, FDA, SEC rules
full_analysis ⚡ 5 A2AMCP Comprehensive analysis with risk scoring and recommendations
scan_video ⚡ 8 A2AMCP Video/audio content compliance scan with transcript analysis
deep_review ⚡ 10 A2AMCP Deep regulatory review with legal citations and action plan

Connect Your Agent

Three ways to integrate: A2A protocol, MCP server, or direct Python SDK.

# Step 1: Discover the agent
curl https://your-server/.well-known/agent.json | jq .

# Step 2: Acquire x402 payment token from Nevermined
TOKEN=$(python -c "
from payments_py import Payments, PaymentOptions
p = Payments.get_instance(PaymentOptions(
    nvm_api_key='YOUR_NVM_API_KEY',
    environment='sandbox'
))
token = p.get_plan_token('PLAN_ID', 1)
print(token.access_token)
")

# Step 3: Send paid compliance scan request
curl -X POST https://your-server/api/v1/a2a \
  -H "Content-Type: application/json" \
  -H "payment-signature: $TOKEN" \
  -d '{
    "jsonrpc": "2.0",
    "method": "tasks/send",
    "params": {
      "message": {
        "role": "user",
        "parts": [{"type":"text","text":"Check: Buy my product, guaranteed results!"}]
      }
    }
  }'
// Claude Desktop config (~/.config/claude/claude_desktop_config.json)
{
  "mcpServers": {
    "compliance-shield": {
      "command": "python",
      "args": ["-m", "src.server"],
      "env": {
        "NVM_API_KEY": "sandbox:your-key",
        "NVM_PLAN_ID": "plan_abc123"
      }
    }
  }
}

// Tools available after connecting:
// - quick_scan (1 credit)    - Fast FTC/FDA/SEC check
// - full_analysis (5 credits) - Comprehensive compliance report
// - scan_video (8 credits)   - Video content analysis
// - deep_review (10 credits)  - Deep regulatory review
from payments_py import Payments, PaymentOptions

# Initialize Nevermined payments client
payments = Payments.get_instance(PaymentOptions(
    nvm_api_key="sandbox:your-api-key",
    environment="sandbox"
))

# Purchase credits for ComplianceShield's plan
order = payments.order_plan("PLAN_ID")
print(f"Order: {order.success}")

# Get an x402 access token (deducts 1 credit)
token = payments.get_plan_token("PLAN_ID", credits=1)

# Call ComplianceShield with payment
import requests
resp = requests.post(
    "https://your-server/api/v1/a2a",
    headers={"payment-signature": token.access_token},
    json={
        "jsonrpc": "2.0",
        "method": "tasks/send",
        "params": {
            "message": {
                "role": "user",
                "parts": [{"type":"text","text":"Scan this for compliance"}]
            }
        }
    }
)
print(resp.json())
