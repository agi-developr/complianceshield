Connect to ComplianceShield

Use compliance scanning from your favorite AI tools — Claude Desktop, MCP clients, autonomous agents, and Telegram.

🤖

Claude Desktop / OpenClaw

Recommended

Connect ComplianceShield as an MCP server in Claude Desktop (or any MCP-compatible client like OpenClaw, Cursor, or Windsurf). Get four compliance tools directly in your chat.

1

Copy the MCP configuration

Add this JSON to your Claude Desktop MCP config file.

claude_desktop_config.json
{ "mcpServers": { "compliance-shield": { "url": "http://localhost:3000/mcp", "headers": { "Authorization": "Bearer YOUR_NVM_API_KEY" } } } }
2

Open Claude Desktop settings

Go to Settings → Developer → MCP Servers and paste the config, or edit the file directly at ~/Library/Application Support/Claude/claude_desktop_config.json

3

Restart Claude Desktop

Close and reopen Claude Desktop. The MCP server will connect automatically.

4

4 compliance tools are now available

You will see these tools listed in the Claude Desktop tool panel:

quick_scan
Fast compliance check (1 credit)
full_analysis
Deep multi-regulation scan (5 credits)
deep_review
Expert legal review (10 credits)
scan_video
YouTube video compliance scan

Example prompts to try:

→ "Scan this text for compliance issues: I lost 30 pounds using this supplement, doctors hate it!"
→ "Run a full_analysis on my product description and check FTC and FDA regulations."
→ "scan_video https://youtube.com/watch?v=example — check for undisclosed sponsorships."

A2A Agent Integration

Connect agent-to-agent using the A2A protocol. Your AI agent can discover ComplianceShield capabilities and purchase scans autonomously.

1

Discover the agent card

Fetch capabilities and pricing from the standard well-known endpoint.

Agent Card URL
http://localhost:9000/.well-known/agent.json
2

Purchase a scan with the Python SDK

Use the payments_py SDK to create a payment token, then send a scan request.

Python (payments_py)
# pip install payments-py httpx from payments_py import Payments, PaymentOptions payments = Payments.get_instance( PaymentOptions( nvm_api_key="YOUR_NVM_API_KEY", environment="sandbox" ) ) # Create access token for the plan token = payments.create_token(plan_id="PLAN_ID", credits=1) # Send scan request via A2A import httpx response = httpx.post( "http://localhost:9000/a2a", headers={"payment-signature": token.access_token}, json={ "jsonrpc": "2.0", "method": "message/send", "params": { "message": { "role": "user", "parts": [{"type": "text", "text": "Scan: Buy now, limited time offer!"}] } } } ) print(response.json())
3

Or use cURL directly

Pass the payment signature as a header for quick testing.

cURL
curl -X POST http://localhost:9000/a2a \ -H "Content-Type: application/json" \ -H "payment-signature: YOUR_ACCESS_TOKEN" \ -d '{ "jsonrpc": "2.0", "method": "message/send", "params": { "message": { "role": "user", "parts": [{"type": "text", "text": "Scan this for FTC compliance: Amazing results guaranteed!"}] } } }'
